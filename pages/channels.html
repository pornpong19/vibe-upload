ช่อง YouTube ที่เชื่อมต่อแล้ว

ยังไม่มีช่องที่เชื่อมต่อ

    วิธีการใช้งาน

    1. ไปที่ Google Cloud Console และสร้างโปรเจค
    2. เปิดใช้งาน YouTube Data API v3
    3. สร้าง OAuth 2.0 Client ID (Desktop app)
    4. เพิ่ม Authorized redirect URI: http://localhost:3000
    5. ดาวน์โหลดไฟล์ credentials.json
    6. อัพโหลดไฟล์ credentials.json ที่นี่และคลิก "เชื่อมต่อและเพิ่มช่อง"
    7. เบราว์เซอร์จะเปิดขึ้นอัตโนมัติ เพื่อให้คุณยืนยันตัวตน
    8. เลือกบัญชี Google และอนุญาตการเข้าถึง
    9. เมื่อเห็นข้อความสำเร็จ ปิดเบราว์เซอร์และกลับมาที่แอป
    10. ช่องจะปรากฏในรายการด้านบนทันที

    สำคัญ! ต้องตั้งค่า Redirect URI เป็น http://localhost:3000 ใน Google Cloud Console ไม่เช่นนั้นจะเชื่อมต่อไม่ได้

    หมายเหตุ: ไฟล์ credentials.json และ tokens ของแต่ละช่องจะถูกเก็บแยกกัน และจะใช้ในการอัพโหลดวิดีโอไปยังช่องนั้นๆ
