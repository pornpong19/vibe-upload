# Assets Directory

โฟลเดอร์นี้สำหรับเก็บไฟล์ static assets เช่น:

- `icon.png` - App icon (สามารถเพิ่มไอคอนของคุณเองได้ที่นี่)

## การเพิ่มไอคอน

1. สร้างไฟล์ `icon.png` ในโฟลเดอร์นี้
2. ขนาดที่แนะนำ: 512x512 pixels หรือ 1024x1024 pixels
3. ไอคอนจะถูกใช้ใน application window

หากไม่มีไฟล์ icon.png, Electron จะใช้ไอคอนเริ่มต้น
